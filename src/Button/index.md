# Button 按钮

[官方原文链接](https://layui.dev/docs/2/button/)

> 向任意 HTML 标签设定 class="layui-btn" 建立一个基础按钮。通过追加格式为 layui-btn-{type}的 class 来定义其它按钮风格。内置的按钮 class 可以进行任意组合，从而形成更多种按钮风格。
